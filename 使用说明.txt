Codex Hypergryph Theme v1.0（macOS / Windows）

macOS：完全退出 Codex，双击“安装终末地主题-macOS.command”。
Windows：安装 Node.js 22 或更高版本，完全退出 Codex，双击“安装终末地主题-Windows.cmd”。脚本可自动识别 PATH、fnm、nvm-windows、Volta 与常见安装目录中的 Node.js。
Windows 启动失败时：双击“检查终末地主题环境-Windows.cmd”执行只读检查。

以后重新启用：双击同系统的“启动终末地主题”。
不再使用主题：双击同系统的“停用主题并恢复官方外观”。

安装不会在桌面或 Windows 开始菜单创建文件。请保留这个解压目录。
Windows 发布包已内置固定版本的 Codex Dream Skin 上游引擎，安装时不需要 Git 或网络下载；本项目只覆盖终末地主题资源与配置。
Windows 会在 %LOCALAPPDATA%\CodexDreamSkin\codex-profile 保存独立的主题会话配置，以兼容新版 Chromium；首次进入时可能需要重新登录，官方 Codex 配置不会被覆盖。
“恢复官方外观”只停止主题注入、关闭本机调试会话并恢复安装前的外观设置；不会删除任务、账号、插件、项目或 Codex。

非官方社区主题。项目名、预览内容均已脱敏。
